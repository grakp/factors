# Factors

The programs calculate the LCM or GCF of two positive integers.

## Build

The following commands build the programs.

### LCM

```
g++ lcm.cpp -o lcm
```

### GCF

```
g++ gcf.cpp -o gcf
```

## Run

The following are examples of computing the LCM or GCF.

### LCM

`Sample Input`
```
$ ./lcm
12 20
```

`Sample Output`
```
60
```

### GCF

`Sample Input`
```
$ ./gcf
9 27
```

`Sample Output`
```
9
```
